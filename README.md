# ğŸ“– My Budget App <a name="about-project"></a>


This project is entails a Ruby on Rails Budget App. It is a web application that uses PostgreSQL for a user to manage his or her budget . It provides a list of transactions associated with a category, so that you can see how much money you spent and on what.
